# Deepin-kwin-settings
This is a program to change some of the settings of deepin-kwin

## How to compile and run
Open a terminal and type the following, \
`mkdir build` \
`cd build` \
`qmake ..` \
`make`

Then just run the compiled `Deepin-kwin-settings` program.

## Requirements
1. Qt (>=5.6)
2. DTKWidget
